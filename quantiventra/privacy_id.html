Kebijakan Privasi

Ray App Lab (selanjutnya disebut sebagai "Pengembang") menetapkan kebijakan privasi berikut mengenai penanganan informasi pengguna dalam aplikasi Android "Quantiventra" (selanjutnya disebut sebagai "Aplikasi").

1. Pengumpulan dan Penggunaan Informasi Pribadi

Aplikasi tidak mengumpulkan atau memperoleh informasi pribadi apa pun seperti nama, alamat, nomor telepon, atau alamat email pengguna. Selain itu, kami tidak mengirimkan informasi apa pun yang diperoleh ke server eksternal atau memberikannya kepada pihak ketiga.

2. Tujuan Penggunaan Izin Perangkat

Aplikasi menggunakan izin berikut semata-mata untuk menyediakan fitur-fiturnya, dan tidak akan menggunakannya untuk tujuan lain apa pun.

  • Kamera (CAMERA): Digunakan secara eksklusif untuk memindai barcode dan kode 2D (termasuk kode QR). Kami tidak merekam, menyimpan, atau mengirimkan foto atau video apa pun ke luar.
  • Penyimpanan (READ/WRITE_EXTERNAL_STORAGE): Digunakan secara eksklusif untuk membaca (mengimpor) dan menulis (mengekspor) file CSV data inventaris. Kami tidak mengakses data apa pun selain file yang ditentukan.

3. Penyimpanan dan Keamanan Data

Semua data yang dimasukkan atau dipindai dalam Aplikasi disimpan secara ketat di perangkat pengguna. Pengembang tidak memiliki akses ke data pengguna. Manajemen data adalah tanggung jawab penuh pengguna.

4. Perubahan Kebijakan Privasi

Pengembang dapat merevisi kebijakan ini karena perubahan undang-undang dan peraturan atau pembaruan aplikasi. Jika ada perubahan signifikan, kami akan mengumumkannya di halaman ini.

5. Hubungi Kami

Untuk pertanyaan mengenai kebijakan ini, silakan hubungi kami melalui informasi kontak di situs web kami atau informasi kontak pengembang yang disediakan di toko aplikasi.
