1. Introdução

1-1. O que é este aplicativo

O Quantiventra é um aplicativo Android para contagem de estoque que usa a câmera do celular para ler códigos de barras e QR codes. Ele ajuda a substituir anotações em papel por registros digitais simples.

Depois de ler um código, você informa a quantidade, adiciona um memorando se necessário e salva. Os resultados podem ser exportados em CSV para Excel, Google Sheets ou sistemas internos.

1-2. Requisitos do sistema

CâmeraObrigatória para leitura
Espaço livreRecomendado 50 MB ou mais

1-3. Permissões

No primeiro início, o app solicita permissão de câmera. Se você negar, a leitura pela câmera não funcionará e será necessário digitar os códigos manualmente.

Armazenamento de dados
O cadastro mestre, o histórico e as configurações são gerenciados automaticamente dentro do app. Para levar dados para fora, use a exportação ou o compartilhamento na tela de Configurações.

2. Instalação e configuração inicial

2-1. Instalação

Instale o app pela Google Play Store e abra o Quantiventra pelo ícone.

2-2. Primeiro uso

  1. Abra o aplicativo.
  2. Permita o uso da câmera.
  3. Se quiser usar um cadastro mestre, abra Configurações e toque em Importar CSV mestre.
  4. Depois disso, você já pode começar a ler, registrar e salvar.

2-3. Como os dados são tratados

No uso normal, não é necessário conhecer a estrutura interna de armazenamento. O usuário só precisa importar um CSV quando quiser carregar o cadastro mestre e exportar ou compartilhar quando quiser retirar dados do app.

O que o usuário faz na prática
Use Importar CSV mestre para carregar o cadastro. Use Exportar todos os dados ou Compartilhar histórico para enviar arquivos para fora do aplicativo.

3. Criação e importação do CSV mestre

Por que vale a pena preparar o mestre
Se o CSV mestre já estiver importado, o nome do item e o estoque teórico aparecem automaticamente após a leitura. Isso reduz digitação manual e evita erros.

3-1. Duas formas de preparar o CSV mestre (com nomes das colunas)

Metodo A: criar do zero no Excel ou Google Sheets

  1. Abra Excel ou Google Sheets no computador.
  2. Nao digite os cabecalhos manualmente. Copie e cole exatamente a linha mostrada em 3-3 Especificacoes das colunas (nomes das colunas).
  3. Preencha os dados a partir da segunda linha.
  4. Salve com o nome quantiventra_master.csv em formato CSV UTF-8.

Metodo B: criar um modelo pelo proprio app

  1. Abra o app e registre um item de exemplo.
  2. Va em Configuracoes e execute Exportar todos os dados.
  3. Envie ou copie o arquivo quantiventra_master_export.csv para o PC.
  4. Abra esse CSV no Excel e use a linha de exemplo como guia. Se voce ja tem uma tabela pronta, cole os dados nas colunas correspondentes.
  5. Salve novamente como CSV UTF-8.
  6. Volte ao app e use Configuracoes -> Importar CSV mestre.
  7. Confirme se nome do item e estoque teorico aparecem corretamente.

3-3. Especificacoes das colunas (nomes das colunas)

A primeira linha e o cabecalho. Copie e cole exatamente a linha abaixo. Os cinco nomes de coluna sao necessarios.

joint_code,barcode,name,theoretical_stock,actual_stock_total

O padrao inicial do app e codigo do parceiro OFF. Se voce usar apenas codigo proprio, deixe a coluna joint_code vazia. Se tambem usar codigo do parceiro, preencha essa coluna. Se houver dados em joint_code, o app liga esse modo automaticamente ao iniciar.
ColunaObrigatorioDescricao

3-5. Como importar o CSV mestre

Voce nao precisa copiar o arquivo para uma pasta fixa. Selecione o arquivo diretamente na tela de configuracoes.

  1. Salve o CSV no celular, no Drive ou envie por email para si mesmo.
  2. No app, abra Configuracoes -> Importar CSV mestre.
  3. Escolha o arquivo no seletor do sistema.
  4. O mestre interno sera substituido pelo arquivo selecionado.
  5. Se editar o arquivo depois, importe novamente a versao atualizada.

5. Fluxo basico de operacao

5-1. Fluxo padrao de inventario

Passo 1: definir localizacao

  1. Toque no botao de leitura de localizacao.
  2. Leia o QR code da prateleira ou digite o local manualmente.
  3. O local continua preenchido ate voce alterar para a proxima prateleira.

Passo 2: ler o codigo do produto

  1. Confirme que a camera esta ligada.
  2. Aponte para o codigo do produto.
  3. Se o item existir no mestre, nome e estoque teorico serao preenchidos.
  4. Se o modo de codigo parceiro estiver ligado e houver ambiguidade, um dialogo pedira para escolher o tipo de codigo.
  5. Se nao existir no mestre, aparecera o dialogo de item nao cadastrado.

Passo 3: informar quantidade

  1. O foco vai automaticamente para Qty.
  2. Digite a quantidade pelo teclado.
  3. Se precisar somar varias contagens, use a calculadora.

Passo 4: salvar

  1. Digite um memo se necessario.
  2. Toque em ENT ou Save.
  3. O app toca um som e o registro entra no historico.

5-2. Usando a calculadora

  1. Toque no campo Calc.
  2. Digite valores como 30, toque em Add, depois 25, Add.
  3. Tambem e possivel usar expressoes como 3*10.
  4. Quando o total estiver correto, salve normalmente.

5-3. Limpar

O botao Clear apaga codigo, nome, quantidade, memo e log da calculadora. O campo de localizacao nao e limpo automaticamente.

6. Operacoes avancadas

6-1. Busca por nome

  1. Ative o modo de busca por nome.
  2. Digite uma palavra-chave no campo do nome.
  3. Toque em Search.
  4. Escolha o item na lista de resultados.
  5. Informe quantidade e salve.

6-2. Busca continua

Quando o modo continuo esta ligado, a busca por nome permanece ativa depois de cada salvamento. Isso ajuda quando varios itens serao registrados por nome no mesmo local.

6-3. Cadastro de novo item

Se um codigo ainda nao existir no mestre, o app mostra um dialogo oferecendo novo cadastro.

  1. Escolha registrar o item.
  2. Se necessario, o app gera um codigo com prefixo 99999.
  3. Digite o nome do item.
  4. Informe a quantidade e salve. O mestre sera atualizado.

6-4. Edicao do nome do item

  1. Leia ou localize o item.
  2. Toque em Edit Name.
  3. Edite o nome no campo correspondente.
  4. Salve para gravar no mestre.

6-5. Edicao do total registrado

  1. Carregue o item.
  2. Toque no botao Edit ao lado de Registered.
  3. Informe o novo valor e aplique.
Atencao
A edicao do total registrado altera somente o cadastro mestre. O historico ja salvo nao e reescrito.

6-6. Escolha entre codigo proprio e codigo parceiro

Quando o modo de codigo parceiro esta ligado, alguns codigos podem ter mais de uma interpretacao. Nesses casos o app mostra um dialogo para evitar abrir o item errado.

10. Solucao de problemas

O codigo nao e lido

  • Confirme que a camera esta ligada.
  • Mantenha o codigo perto do centro da tela.
  • Verifique se o formato correto esta habilitado nas configuracoes.
  • Se o codigo estiver danificado, use digitacao manual.
  • Confirme a permissao de camera nas configuracoes do Android.

O nome do item nao aparece apos importar o CSV

  • Verifique se o arquivo selecionado era o correto.
  • Confirme o cabecalho: joint_code,barcode,name,theoretical_stock,actual_stock_total.
  • Garanta que o arquivo esteja em UTF-8.
  • Se quiser trocar para outro CSV, execute Importar CSV mestre novamente em Configurações e selecione o arquivo.

Falha ao salvar

  • Confirme se existe codigo ou nome preenchido.
  • Verifique se ha espaco livre no dispositivo.

A leitura de localizacao nao define a prateleira

  • Veja se o formato de leitura de localizacao esta correto.
  • Se a etiqueta usar outro padrao, ajuste nas configuracoes.

Nao encontro os arquivos exportados

Procure na pasta escolhida durante a exportacao. Os nomes padrao sao scan_history_[timestamp].csv e quantiventra_master_export.csv.

O campo Calc nao aplica a conta

  • Use Add para somar o resultado em Qty.
  • Evite caracteres de largura total.
  • A expressao nao pode comecar por operador, como *5.
