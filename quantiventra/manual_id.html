1. Pendahuluan

1-1. Tentang aplikasi ini

Quantiventra adalah aplikasi Android untuk menghitung stok dengan bantuan kamera ponsel. Aplikasi ini membaca barcode atau QR code, lalu membantu Anda mencatat jumlah secara cepat.

Setelah kode dibaca, Anda memasukkan jumlah, menambahkan memo bila perlu, lalu menyimpan. Data dapat diekspor sebagai CSV untuk Excel, Google Sheets, atau sistem lain.

1-2. Persyaratan sistem

Sistem operasiAndroid 7.0 (API 24) atau lebih baru
KameraWajib untuk pemindaian
Ruang kosongDisarankan 50 MB atau lebih
IzinKamera
InternetTidak wajib

1-3. Izin aplikasi

Saat pertama kali dijalankan aplikasi meminta izin kamera. Jika izin ditolak, pemindaian tidak bisa digunakan dan Anda harus memasukkan kode secara manual.

Penyimpanan data
Data master, riwayat, dan pengaturan dikelola otomatis di dalam aplikasi. Untuk membawa data keluar, gunakan ekspor atau berbagi dari layar pengaturan.

2. Instalasi dan pengaturan awal

2-1. Instalasi

Instal aplikasi dari Google Play Store lalu buka Quantiventra dari ikonnya.

2-2. Penggunaan pertama

  1. Buka aplikasi.
  2. Izinkan penggunaan kamera.
  3. Jika ingin memakai data master, buka Pengaturan dan pilih Impor CSV master.
  4. Setelah itu Anda dapat mulai memindai, mencatat, dan menyimpan.

2-3. Cara data ditangani

Dalam penggunaan normal Anda tidak perlu mengetahui struktur penyimpanan internal. Pengguna hanya perlu mengimpor CSV saat ingin memuat master dan mengekspor atau berbagi saat ingin mengeluarkan data dari aplikasi.

Apa yang perlu dilakukan pengguna
Gunakan Impor CSV master untuk memuat data master. Gunakan Ekspor semua data atau Bagikan riwayat saat ingin mengambil file dari aplikasi.

3. Membuat dan mengimpor CSV master

Manfaat menyiapkan master
Jika CSV master sudah diimpor, nama barang dan stok teoritis akan tampil otomatis setelah pemindaian. Ini mengurangi pengetikan manual dan membantu mencegah kesalahan.

3-1. Dua cara menyiapkan CSV master (dengan nama kolom)

Metode A: membuat dari nol di Excel atau Google Sheets

  1. Buka Excel atau Google Sheets di PC.
  2. Jangan mengetik header secara manual. Salin dan tempel persis baris yang ada di 3-3 Spesifikasi kolom (nama kolom).
  3. Isi data mulai dari baris kedua.
  4. Simpan file dengan nama quantiventra_master.csv dalam format CSV UTF-8.

Metode B: membuat template dari aplikasi

  1. Buka aplikasi dan daftarkan satu item contoh.
  2. Masuk ke Pengaturan lalu jalankan Ekspor semua data.
  3. Kirim atau salin file quantiventra_master_export.csv ke PC.
  4. Buka file itu di Excel dan gunakan baris contoh sebagai panduan. Jika Anda sudah punya tabel barang, tempelkan data ke kolom yang sesuai.
  5. Simpan kembali sebagai CSV UTF-8.
  6. Kembali ke aplikasi lalu gunakan Pengaturan -> Impor CSV master.
  7. Pastikan nama barang dan stok teoritis tampil dengan benar.

3-2. Nama file dan persyaratan

ItemKeterangan
Nama filequantiventra_master.csv
EncodingUTF-8 disarankan
Baris baruLF atau CRLF
ImporDari Pengaturan -> Impor CSV master

3-3. Spesifikasi kolom (nama kolom)

Baris pertama adalah header. Salin dan tempel persis satu baris berikut. Kelima nama kolom wajib ada.

joint_code,barcode,name,theoretical_stock,actual_stock_total
Walaupun kode partner masih OFF, tetap gunakan 5 kolom yang sama
Pengaturan awal aplikasi adalah kode partner OFF. Jika Anda hanya memakai kode sendiri, biarkan kolom joint_code kosong. Jika Anda juga memakai kode partner, isi kolom tersebut. Bila master yang diimpor berisi data pada joint_code, aplikasi akan menyalakan mode itu secara otomatis saat dibuka.
KolomWajibKeterangan
joint_codeTidakKode partner atau pelanggan.
barcodeYaKode sendiri atau barcode. Harus unik.
nameYaNama barang. Jika ada koma, gunakan tanda kutip ganda.
theoretical_stockTidakStok teoritis. Gunakan 0 jika tidak diketahui.
actual_stock_totalTidakTotal tercatat yang sudah terkumpul. Untuk inventaris pertama biasanya 0.

Penting tentang encoding
Simpan selalu dalam UTF-8. Format lain dapat menyebabkan karakter rusak.

3-5. Cara mengimpor CSV master

Anda tidak perlu menaruh file di folder tertentu. Pilih langsung dari layar pengaturan.

  1. Simpan CSV di ponsel, Drive, atau kirim lewat email.
  2. Di aplikasi buka Pengaturan -> Impor CSV master.
  3. Pilih file di pemilih file sistem.
  4. Master internal akan diganti dengan file yang dipilih.
  5. Jika Anda mengedit file lagi nanti, impor ulang versi terbarunya.

Aplikasi memprioritaskan kode yang berada dekat pusat layar. Usahakan kode tetap berada di tengah.

5. Alur kerja dasar

5-1. Alur inventaris standar

Langkah 1: tentukan lokasi

  1. Tekan tombol pemindaian lokasi.
  2. Pindai QR code rak atau isi lokasi secara manual.
  3. Lokasi tetap tersimpan sampai Anda menggantinya.

Langkah 2: pindai kode produk

  1. Pastikan kamera aktif.
  2. Arahkan ke kode produk.
  3. Jika barang ada di master, nama dan stok teoritis akan terisi.
  4. Jika mode kode partner aktif dan terjadi ambiguitas, aplikasi menampilkan dialog pilihan.
  5. Jika kode tidak ada di master, akan muncul dialog untuk mendaftarkan barang baru.

Langkah 3: masukkan jumlah

  1. Fokus otomatis berpindah ke Qty.
  2. Masukkan jumlah dengan keypad.
  3. Jika perlu menjumlahkan beberapa hitungan, gunakan kalkulator.

Langkah 4: simpan

  1. Tambahkan memo bila perlu.
  2. Tekan ENT atau Save.
  3. Aplikasi memutar bunyi dan data masuk ke riwayat.

5-2. Menggunakan kalkulator

  1. Ketuk kolom Calc.
  2. Masukkan angka seperti 30, lalu Add, kemudian 25, Add.
  3. Anda juga bisa memakai ekspresi seperti 3*10.
  4. Jika total sudah benar, simpan seperti biasa.

5-3. Membersihkan layar

Tombol Clear menghapus kode, nama, jumlah, memo, dan log kalkulator. Kolom lokasi tidak dihapus otomatis.

6. Operasi lanjutan

6-1. Pencarian berdasarkan nama

  1. Aktifkan mode pencarian nama.
  2. Ketik kata kunci di kolom nama barang.
  3. Tekan Search.
  4. Pilih barang dari daftar hasil.
  5. Masukkan jumlah lalu simpan.

6-2. Pencarian berkelanjutan

Jika mode berkelanjutan aktif, pencarian nama tetap menyala setelah setiap penyimpanan. Ini berguna jika Anda mencatat banyak barang berdasarkan nama di lokasi yang sama.

6-3. Pendaftaran barang baru

Jika kode yang dipindai belum ada di master, aplikasi menampilkan dialog pendaftaran baru.

  1. Pilih untuk mendaftarkan barang.
  2. Jika perlu, aplikasi akan membuat kode dengan awalan 99999.
  3. Masukkan nama barang.
  4. Masukkan jumlah lalu simpan. Data master akan diperbarui.

6-4. Mengedit nama barang

  1. Muat barang dengan pemindaian atau pencarian.
  2. Tekan Edit Name.
  3. Ubah nama pada kolom yang tersedia.
  4. Simpan untuk memperbarui master.

6-5. Mengedit total tercatat

  1. Muat barangnya.
  2. Tekan Edit di samping Registered.
  3. Masukkan nilai baru lalu terapkan.
Perhatian
Pengeditan total tercatat hanya mengubah file master. Riwayat yang sudah tersimpan tidak ditulis ulang.

6-6. Memilih kode sendiri atau kode partner

Jika mode kode partner aktif, beberapa kode bisa memiliki lebih dari satu arti. Dalam keadaan itu aplikasi menampilkan dialog pilihan agar barang yang terbuka tidak salah.

7. Riwayat

Bagian bawah layar menampilkan catatan terbaru. Tombol History membuka daftar lengkap yang berisi tanggal, waktu, nama, lokasi, memo, jumlah, dan stok teoritis.

8-2. Hapus semua riwayat

Menghapus scan_history.csv dan mengosongkan riwayat.

10. Pemecahan masalah

Kode tidak terbaca

  • Pastikan kamera aktif.
  • Jaga kode berada dekat tengah layar.
  • Periksa apakah format yang benar aktif di pengaturan.
  • Jika kode rusak, gunakan input manual.
  • Periksa izin kamera di Android.

Nama barang tidak muncul setelah impor CSV

  • Pastikan Anda memilih file yang benar.
  • Periksa header: joint_code,barcode,name,theoretical_stock,actual_stock_total.
  • Pastikan file menggunakan UTF-8.
  • Jika ingin memakai CSV lain, jalankan Impor CSV master lagi dari Pengaturan lalu pilih file tersebut.

Gagal menyimpan

  • Pastikan ada kode atau nama barang yang terisi.
  • Periksa ruang kosong perangkat.

Pemindaian lokasi tidak mengisi rak

  • Periksa format pemindaian lokasi.
  • Jika label memakai format lain, ubah di pengaturan.

Saya tidak menemukan file ekspor

Cari di folder yang dipilih saat ekspor. Nama umumnya adalah scan_history_[timestamp].csv dan quantiventra_master_export.csv.

Kalkulator tidak menerapkan hasil

  • Gunakan Add untuk menambahkan hasil ke Qty.
  • Hindari karakter full-width.
  • Ekspresi tidak boleh dimulai dengan operator, misalnya *5.

11. Ketentuan dan penafian

Dengan menggunakan Quantiventra Anda dianggap menyetujui ketentuan berikut.

Lisensi: aplikasi dapat digunakan untuk inventaris dan perhitungan stok oleh individu atau perusahaan. Rekayasa balik, distribusi ulang tanpa izin, dan modifikasi yang tidak sah dilarang.

Data pengguna: file yang dibuat aplikasi tetap berada di perangkat pengguna. Aplikasi tidak mengirim data ke server eksternal.

Penggunaan terlarang: aplikasi tidak boleh digunakan untuk tujuan ilegal, penipuan, atau pelanggaran hak pihak ketiga.

Penafian: aplikasi disediakan sebagaimana adanya. Pengembang tidak bertanggung jawab atas kehilangan data, gangguan bisnis, atau kerugian langsung maupun tidak langsung akibat penggunaan aplikasi.

Privasi: tidak ada informasi pribadi yang dikumpulkan. Kamera hanya digunakan untuk membaca kode. Tidak ada video yang direkam atau dikirim.
