1. Introducción

1-1. Qué es esta aplicación

Quantiventra es una aplicación Android para conteo de inventario que usa la cámara del teléfono para leer códigos de barras y códigos QR. Su objetivo es simplificar el conteo físico y convertirlo en registros digitales fáciles de exportar.

Después de leer un código, usted ingresa la cantidad, agrega una nota si lo necesita y guarda. Los resultados pueden exportarse como CSV para Excel, Google Sheets u otros sistemas.

Sistema operativoAndroid 7.0 (API 24) o superior
CámaraObligatoria para el escaneo
Espacio libreSe recomiendan 50 MB o más

1-3. Permisos

En el primer inicio, la aplicación solicita permiso de cámara. Si usted lo rechaza, no podrá usar el escaneo y tendrá que ingresar los códigos manualmente.

Almacenamiento de datos
El maestro, el historial y la configuración se administran automáticamente dentro de la aplicación. Para sacar una copia, use Exportar o Compartir desde la pantalla de Configuración.

2. Instalación y configuración inicial

2-1. Instalación

Instale la aplicación desde Google Play Store y abra Quantiventra desde su ícono.

2-2. Primer uso

  1. Abra la aplicacion.
  2. Permita el uso de la cámara.
  3. Si desea trabajar con un maestro, abra Configuración y seleccione Importar CSV maestro.
  4. Después de eso ya puede empezar a escanear, registrar y guardar.

2-3. Cómo se manejan los datos

En el uso normal no es necesario conocer la estructura interna del almacenamiento. El usuario solo necesita importar un CSV cuando quiera cargar un maestro y exportar o compartir cuando quiera sacar datos de la aplicación.

Qué debe hacer el usuario
Use Importar CSV maestro para cargar datos maestros. Use Exportar todos los datos o Compartir historial para sacar archivos de la aplicación.

3. Creación e importación del CSV maestro

Ventaja de preparar el maestro
Cuando el CSV maestro ya está importado, el nombre del artículo y el stock teórico aparecen automáticamente después del escaneo. Esto ahorra tiempo y reduce errores.

3-1. Dos formas de preparar el CSV maestro (con nombres de columna)

Metodo A: crear desde cero en Excel o Google Sheets

  1. Abra Excel o Google Sheets en su PC.
  2. No escriba manualmente los encabezados. Copie y pegue exactamente la linea mostrada en 3-3 Especificacion de columnas (nombres de columna).
  3. Ingrese los datos a partir de la segunda fila.
  4. Guarde el archivo como quantiventra_master.csv en formato CSV UTF-8.

Metodo B: crear una plantilla desde la app

  1. Abra la app y registre un articulo de ejemplo.
  2. Entre a Configuracion y ejecute Exportar todos los datos.
  3. Envie o copie el archivo quantiventra_master_export.csv a su PC.
  4. Abra ese archivo en Excel y use la fila de ejemplo como guia. Si ya tiene una tabla preparada, pegue sus datos en las columnas correspondientes.
  5. Guarde de nuevo como CSV UTF-8.
  6. Regrese a la app y use Configuracion -> Importar CSV maestro.
  7. Verifique que el nombre del articulo y el stock teorico aparezcan correctamente.

3-2. Nombre del archivo y requisitos

ElementoDescripcion
Nombre del archivoquantiventra_master.csv

3-3. Especificacion de columnas (nombres de columna)

La primera fila es el encabezado. Copie y pegue exactamente la linea siguiente. Los cinco nombres de columna son obligatorios.

joint_code,barcode,name,theoretical_stock,actual_stock_total
Aunque el codigo del cliente este desactivado, siga usando las 5 columnas
La configuracion inicial es codigo del cliente OFF. Si trabaja solo con su propio codigo, deje vacia la columna joint_code. Si tambien usa codigo del cliente, rellene esa columna. Si el maestro importado contiene datos en joint_code, la aplicacion activa ese modo automaticamente al iniciar.
ColumnaObligatoriaDescripcion
joint_codeNoCodigo del cliente o socio comercial.
barcodeSiCodigo propio o codigo de barras. Debe ser unico.
nameSiNombre del articulo. Si contiene comas, use comillas dobles.
theoretical_stockNoStock teorico. Use 0 si no lo conoce.
actual_stock_totalNoTotal acumulado registrado. Para el primer inventario, normalmente 0.

Importante sobre la codificacion
Guarde siempre en UTF-8. Otros formatos pueden provocar caracteres dañados.

3-5. Como importar el CSV maestro

No necesita copiar el archivo a una carpeta fija. Seleccionelo directamente desde la pantalla de configuracion.

  1. Guarde el CSV en el telefono, en Drive o envielo por correo.
  2. En la app abra Configuracion -> Importar CSV maestro.
  3. Seleccione el archivo en el selector del sistema.
  4. El maestro interno se reemplazara con el archivo elegido.
  5. Si vuelve a editar el archivo mas adelante, importe otra vez la version actualizada.

5. Flujo basico de uso

5-1. Flujo estandar de inventario

Paso 1: definir la ubicacion

  1. Toque el boton de lectura de ubicacion.
  2. Lea el QR del estante o escriba la ubicacion manualmente.
  3. La ubicacion permanece hasta que usted la cambie.

Paso 2: leer el codigo del producto

  1. Confirme que la camara este activada.
  2. Apunte al codigo del producto.
  3. Si el articulo existe en el maestro, se completan nombre y stock teorico.
  4. Si el modo de codigo del cliente esta activo y hay ambiguedad, aparecera un dialogo de seleccion.
  5. Si el codigo no existe en el maestro, aparecera el dialogo para registrar un nuevo articulo.

Paso 3: ingresar cantidad

  1. El foco pasa automaticamente a Qty.
  2. Ingrese la cantidad con el teclado.
  3. Si necesita sumar varias cantidades, use la calculadora.

Paso 4: guardar

  1. Agregue un memo si es necesario.
  2. Toque ENT o Save.
  3. La app reproduce un sonido y el registro entra al historial.

5-2. Uso de la calculadora

  1. Toque el campo Calc.
  2. Ingrese valores como 30, Add, luego 25, Add.
  3. Tambien puede usar expresiones como 3*10.
  4. Cuando el total sea correcto, guarde normalmente.

5-3. Limpiar

El boton Clear borra codigo, nombre, cantidad, memo y el log de calculo. La ubicacion no se borra automaticamente.

6. Operaciones avanzadas

6-1. Busqueda por nombre

  1. Active el modo de busqueda por nombre.
  2. Escriba una palabra clave en el campo del nombre.
  3. Toque Search.
  4. Seleccione el articulo en la lista.
  5. Ingrese cantidad y guarde.

6-2. Busqueda continua

Cuando el modo continuo esta activado, la busqueda por nombre sigue habilitada despues de cada guardado. Es util para registrar varios articulos por nombre en la misma ubicacion.

6-3. Registro de articulos nuevos

Si se escanea un codigo que todavia no esta en el maestro, la aplicacion muestra un dialogo de nuevo registro.

  1. Elija registrar el articulo.
  2. Si hace falta, la app genera un codigo con prefijo 99999.
  3. Ingrese el nombre del articulo.
  4. Ingrese la cantidad y guarde. El maestro se actualizara.

6-4. Edicion del nombre del articulo

  1. Cargue el articulo mediante lectura o busqueda.
  2. Toque Edit Name.
  3. Edite el nombre en el campo correspondiente.
  4. Guarde para actualizar el maestro.

6-5. Edicion del total registrado

  1. Cargue el articulo.
  2. Toque Edit junto a Registered.
  3. Ingrese el nuevo valor y confirme.
Atencion
La edicion del total registrado solo modifica el maestro. El historial ya guardado no se reescribe.

6-6. Eleccion entre codigo propio y codigo del cliente

Cuando el modo de codigo del cliente esta activado, algunos codigos pueden coincidir con mas de un campo. En ese caso la app muestra un dialogo para evitar abrir el articulo equivocado.

La barra inferior muestra los registros mas recientes. El boton History abre la lista completa con fecha, hora, nombre, ubicacion, memo, cantidad y stock teorico.

8. Configuracion

Abra la configuracion desde el icono superior.

8-1. Importar CSV maestro

Seleccione un CSV desde el dispositivo, la carpeta Descargas o la nube para actualizar el maestro interno. La importación se aplica de inmediato, así que no hace falta ejecutar una recarga aparte.

8-2. Borrar todo el historial

Elimina scan_history.csv y reinicia el historial.

Esta accion no se puede deshacer
Exporte los datos antes de borrar si quiere conservar una copia.

8-3. Usar codigo del cliente

Cuando esta opcion esta activa, la app tambien trabaja con el campo de codigo del cliente.

DesactivadoSolo se usa codigo propio

8-7. Idioma y manual

La aplicacion admite japones, ingles, chino simplificado, chino tradicional, coreano, hindi, vietnamita, italiano, portugues de Brasil, espanol de Latinoamerica, indonesio y tailandes.

Despues de cambiar el idioma, toque Save en la pantalla de configuracion y vuelva a abrir el manual.

10. Solucion de problemas

El codigo no se escanea

  • Confirme que la camara este activada.
  • Mantenga el codigo cerca del centro de la pantalla.
  • Revise que el formato correcto este habilitado en configuracion.
  • Si el codigo esta dañado, use captura manual.
  • Verifique el permiso de camara en Android.

El nombre del articulo no aparece despues de importar el CSV

  • Compruebe que selecciono el archivo correcto.
  • Revise el encabezado: joint_code,barcode,name,theoretical_stock,actual_stock_total.
  • Asegure que el archivo este en UTF-8.
  • Si quiere cambiar a otro CSV, ejecute Importar CSV maestro otra vez en Configuración y seleccione el archivo.

Fallo al guardar

  • Confirme que exista un codigo o un nombre ingresado.
  • Revise que el dispositivo tenga espacio libre.

La lectura de ubicacion no fija el estante

  • Verifique el formato de lectura de ubicacion.
  • Si su etiqueta usa otro tipo de codigo, cambielo en configuracion.

No encuentro los archivos exportados

Busquelos en la carpeta elegida durante la exportacion. Los nombres habituales son scan_history_[timestamp].csv y quantiventra_master_export.csv.

La calculadora no aplica el resultado

  • Use Add para sumar el resultado a Qty.
  • Evite caracteres de ancho completo.
  • La expresion no puede empezar con un operador, por ejemplo *5.
